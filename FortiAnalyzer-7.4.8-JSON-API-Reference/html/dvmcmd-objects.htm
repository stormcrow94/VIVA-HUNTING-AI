add/dev-list (command)

Add multiple devices to the Device Manager database.

Supported methods: exec

Object URL:
Command object/dvm/cmd/add/dev-list
Attribute ListDescriptions
add-dev-listtable
A list of device objects to be added. Refer to device action attribute to select different type of add operation.
adm_passpassword (128 bytes)
add real and promote device.
adm_usrstring (36 bytes)
add real and promote device.
descstring (128 bytes)
available for all operations.
device actionstring
Specify add device operations, or leave blank to add real device:
  • "add_model" - add a model device.
  • "promote_unreg" - promote an unregistered device to be managed by FortiManager using information from database.
faz.quotaunsigned int32
available for all operations.

adomdatasource, refer to adom object (maximum 1) (36 bytes)

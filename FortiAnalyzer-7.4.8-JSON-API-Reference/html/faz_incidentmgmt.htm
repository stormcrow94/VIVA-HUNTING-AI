Add an Incident

add /incidentmgmt/adom/<adom-name>/incident

Add a new incident.

Request:
{
    "method": "add",
    "params": [
        {
            "url": "/incidentmgmt/adom/<adom-name>/incident",
            "reporter": "string",
            "epid": number,
            "euid": number,
            "endpoint": "string",
            "category": "string",
            "severity": "string",
            "status": "string",
            "description": "string",
            "apiver": 3
        }
    ],
    "jsonrpc": "2.0",
    "id": "..."
}

Start Log Search Request

add /logview/adom/<adom-name>/logsearch

Start a new task to search logs.

Request:
{
    "method": "add",
    "params": [
        {
            "url": "/logview/adom/<adom-name>/logsearch",
            "apiver": 3,
            "device": [
                {"devname": "string"},
                {"devid": "string"}
            ],
            "logtype": "string",
            "filter": "string",
            "time-order": "string",
            "time-range": {"start": "string", "end": "string"},
            "case-sensitive": boolean
        }
    ],
    "jsonrpc": "2.0",
    "session": "...",
    "id": "..."
}

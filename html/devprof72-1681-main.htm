System Template (For v7.2)


This documentation applies to v7.2 devices or ADOM, latest release (build 1681).

System Template manages common system configurations that can be shared among various devices. All configurations under this template are device global objects. Once a template is assigned to a device, all configuration will be copied onto the device during the next install.

  • Prefix /pm/devprof/adom/ is used to access or modify the template and assignment.
  • Prefix /pm/config/adom/devprof/ is used to access objects managed by the template.

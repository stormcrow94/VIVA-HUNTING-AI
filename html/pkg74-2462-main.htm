Policy Package (For v7.4 Older Releases)


This documentation applies to v7.4 devices or ADOM, older release (build 2462).

Policy package manages the different types of policies in device VDOM. All configuration will be copied onto the device during next install.

  • Prefix /pm/pkg/global/ and /pm/pkg/adom/ are used to access or modify the packages and install targets at global or ADOM level.
  • Prefix /pm/config/global/pkg/ and /pm/config/adom/pkg/ are used to access policies inside a package.
  • Prefix /pm/pblock/adom/ and /pm/config/adom/pblock/ are use to configure policy block and policy inside policy block.

This list only contains the policies. All objects that are reference by the policies are shared among the packages, and are listed under ADOM Level Objects.

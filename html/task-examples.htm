Task Database Examples


Task database query example


Query the summary of all security console install packages (src = 1) tasks that are contains errors (state = 5) or warnings (state = 8).

Request:
{
    "id": 1,
    "method": "get",
    "params": [ 
        {
            "url": "/task/task",
            "fields": [ "id", "title", "state" ],
            "filter": [ 
                [ 
                    [ "state", "==", 5 ],
                    "||",
                    [ "state", "==", 8 ] 
                ],
                "&&",
                [ "src", "==", 1 ] 
            ],
            "sortings": [ 
                {
                    "id": -1 
                } 
            ],
            "loadsub": 0 
        } 
    ] 
}

Result:
{
    "id": 1,
    "result": [ 
        {
            "data": [ 
                {
                    "id": 434,
                    "state": 5,
                    "title": "Reinstall packages" 
                },
                {
                    "id": 388,
                    "state": 8,
                    "title": "Reinstall packages" 
                },
                {
                    "id": 386,
                    "state": 8,
                    "title": "Copy Package 'default'" 
                },
                {
                    "id": 385,
                    "state": 8,
                    "title": "Reinstall packages" 
                },
                {
                    "id": 380,
                    "state": 5,
                    "title": "Reinstall packages" 
                },
                ... 
            ],
            "url": "/task/task" 
        } 
    ] 
}

Query the history of a specific task line using name and VDOM filters.

Request:
{
    "id": 4,
    "method": "get",
    "params": [ 
        {
            "url": "/task/task/388/history",
            "filter": [ 
                [ "name", "==", "fg1[copy]" ],
                "&&",
                [ "vdom", "==", "root" ] 
            ],
            "sortings": [ 
                {
                    "percent": 1 
                } 
            ] 
        } 
    ] 
}

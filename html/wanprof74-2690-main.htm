SD-WAN Template (For v7.4)


This documentation applies to v7.4 devices or ADOM, latest release (build 2690).

SD-WAN Template manages a subset of configuration of system virtual-wan-link or system sdwan object at device VDOM level. Once a template is assigned to a device, all configuration will be copied onto the device during the next install.

  • Prefix /pm/wanprof/adom/ is used to access or modify the template and assignment.
  • Prefix /pm/config/adom/wanprof/ is used to access the system virtual-wan-link or system sdwan objects managed.

All objects that are reference by the template are shared, and are listed under ADOM Level Objects.
